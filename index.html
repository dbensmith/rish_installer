Shizuku Rish Installer for Termux

Fork of merbah3266/rish_installer with Termux and thedjchi/Shizuku support.

Install

curl -fsSL https://raw.githubusercontent.com/dbensmith/rish_installer/main/rish_installer.sh | bash

Uninstall

curl -fsSL https://raw.githubusercontent.com/dbensmith/rish_installer/main/rish_installer.sh | bash -s -- --uninstall

What changed

  • Preferred tool fallback order: native tools → installed busybox → Termux package install → downloaded BusyBox
  • Preferred release fallback order: thedjchi/Shizuku → RikkaApps/Shizuku
  • All URLs now point to dbensmith/rish_installer

If you publish a short URL, make sure it redirects to this fork’s raw script URL, not the upstream repository.
