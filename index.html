Shizuku Rish Installer for Termux

Fork of merbah3266/rish_installer with Termux and thedjchi/Shizuku support. No bundled BusyBox binaries.

Install

curl -fsSL https://raw.githubusercontent.com/dbensmith/rish_installer/main/rish_installer.sh | bash

Uninstall

curl -fsSL https://raw.githubusercontent.com/dbensmith/rish_installer/main/rish_installer.sh | bash -s -- --uninstall

Missing tools?

If the script exits with Required tools missing, install BusyBox first:

pkg install busybox

What changed

  • Tool fallback order: native tools → installed busybox → Termux pkg/apt install busybox → clear error (no binary download)
  • Release fallback order: thedjchi/Shizuku → RikkaApps/Shizuku
  • No bundled BusyBox binaries in this repo

If you publish a short URL, make sure it redirects to this fork’s raw script URL, not the upstream repository.
